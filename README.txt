An investigation into the 2018 United States senate election that compares
voter demographics from 2012-2016 with the results of the election
for each county. 

Technologies used: Python, plotly, geopands, shapely, 
		   pyshp, pandas, numpy, seaborn, scipy

In order to run the code, make sure you have all this installed:
	Python3
	Jupyter Notebooks
	Python Libraries: plotly, geopands, shapely, pyshp, pandas, numpy, seaborn, scipy
	
Then open the file 'Project_Code.ipynb' in Jupyter Notebooks.
